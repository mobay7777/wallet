# Wallet - Secure Wallet for Rupaya
Wallet web version can send RUPX, RRC20, RRC21. It can help you manage your balance and transactions also.

Support both **Rupaya Mainet** and and **Rupaya testnet**

## Requirements
Node v8.9.0

## Install
`npm install`

## Run
`npm start`

## Build
`npm run build`
